# 参与反馈

> 最新书稿已更新至 [XQuant 量化课堂页](https://xquant.shop/courses)。
> 想阅读最新版官方书稿，请前往图书页。

感谢你阅读《XQuant：人人都是量化交易员》开源版。

这个仓库主要收集书稿反馈，帮助正式书继续修订。最有价值的反馈不是
“这里不好”，而是能定位到具体章节、具体段落、具体问题。

## 反馈入口

请优先使用 GitHub Issue：

`https://github.com/xingwudao/xquant-beginner/issues/new/choose`

如果你只是想加入读者群，可以添加作者微信：

`xquanter`

请备注：

`XQuant 读者`

## 适合提交的反馈

- 错别字、标点、排版问题
- 某个概念没看懂
- 某段叙述前后不一致
- 书中实验结果和你运行出的结果不一致
- 对正式书结构、案例、附录的建议

## 不适合提交的内容

- 个股、ETF、策略买卖建议
- 让作者判断某个投资机会是否值得买
- 和本书无关的工具安装问题
- 没有上下文的截图或一句话评论

## 提交 Issue 时请写清楚

- 章节名称
- 小节标题
- 原文片段
- 你遇到的问题
- 你建议如何修改

## Pull Request

欢迎提交小范围 PR，尤其是错别字和链接修复。

PR 请保持聚焦：

- 一个 PR 只解决一类问题
- 不要混合改多个章节的风格
- 不要批量重写语气，除非先开 Issue 讨论
- 不要提交 `docs/`、`node_modules/`、`.vitepress/dist/`
